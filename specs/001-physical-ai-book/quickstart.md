# Quickstart Guide

This quickstart guide will provide essential steps to get started with the Physical AI Book project.

## 1. Environment Setup

*   **Operating System**: Ubuntu 22.04 LTS (recommended)
*   **Dependencies**: Ensure `git`, `Node.js` (for Docusaurus), and `Python 3.10+` are installed.

## 2. Clone the Repository

```bash
git clone [repository_url]
cd physical-ai-book
```

## 3. Install Docusaurus Dependencies

```bash
npm install
```

## 4. Start Docusaurus Development Server

```bash
npm run start
```

This will launch the documentation site locally.

## 5. Explore the Content

Navigate through the `docs` directory to explore the modules and chapters.

## 6. Running Code Examples

Refer to the `code-examples` directory for specific instructions on setting up and running the robot code.
