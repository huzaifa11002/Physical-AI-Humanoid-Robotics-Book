# Additional Resources

This section provides a curated list of external resources to deepen your understanding and explore advanced topics related to Physical AI and humanoid robotics.

## Books and Publications

*   **"Robotics, Vision and Control" by Peter Corke**: A comprehensive textbook covering robotics fundamentals, kinematics, dynamics, and control, with practical examples in MATLAB and Python.
*   **"Probabilistic Robotics" by Sebastian Thrun, Wolfram Burgard, Dieter Fox**: A foundational text on mobile robot localization, mapping, and navigation using probabilistic methods.
*   **"Deep Learning for Robotics" (Various Authors)**: A collection of research papers and articles exploring the application of deep learning to robotic perception, control, and reinforcement learning.

## Online Courses and Tutorials

*   **ROS 2 Documentation**: The official documentation for ROS 2 provides extensive tutorials, concepts, and API references. [Link to ROS 2 Docs]
*   **NVIDIA Isaac Sim Tutorials**: NVIDIA's official tutorials for Isaac Sim cover basic usage, asset creation, and advanced simulation techniques. [Link to Isaac Sim Docs]
*   **Unity Robotics Tutorials**: Official Unity tutorials for integrating robotics simulations with ROS and other tools. [Link to Unity Robotics Docs]
